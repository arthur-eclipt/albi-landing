# Albi&Us landing page

Static landing page for albiandus.com. Single `index.html` with no build step, no dependencies, no JS.

## Deploy

Upload `index.html` to any static host (Cloudflare Pages, Netlify, Vercel, GitHub Pages, S3+CloudFront, plain Nginx). Point `albiandus.com` at it. Configure `albiandus.co.uk` to 301-redirect to `albiandus.com` at the DNS/host level. The footer links to `/privacy` and `/terms` — those pages are added separately.
